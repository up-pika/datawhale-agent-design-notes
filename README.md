# Agent 设计模式 · 课程笔记

本仓库用于开源分享 **Agent 设计模式** 相关课程的学习笔记、要点整理与实践记录。

## 官方教程（DataWhale）

- **教程入口**：[DataWhale · 学用 AI — 本课学习页](https://www.datawhale.cn/activity/542/learn/297/6412)  
- 笔记与官方章节并非一一强制对应：建议以平台上的目录为准，在 [`notes/00-outline.md`](notes/00-outline.md) 里维护自己的进度表。

## 内容说明

- `notes/`：按主题或章节组织的 Markdown 笔记
- 文前/导读整理见 [`notes/01-preface-and-intro.md`](notes/01-preface-and-intro.md)
- 欢迎通过 Issue / PR 指正与补充（若你 fork 自用，也可仅作个人归档）

## 本地使用

```bash
git clone <你的仓库地址>
cd datawhale
```

在 `notes/` 下按自己的节奏新增或修改 `.md` 文件即可。
